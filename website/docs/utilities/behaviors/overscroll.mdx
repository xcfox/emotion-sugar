# Overscroll Behavior

Utilities for setting what a browser does when reaching the boundary of a scrolling area.

#### [MDN Reference](https://developer.mozilla.org/docs/Web/CSS/overscroll-behavior)

## Utilities

| Utilities                  | CSS Properties                                              |
| -------------------------- | ----------------------------------------------------------- |
| sugar().overscroll(value)  | overscroll-behavior: {value};                               |
| sugar().overscroll(x, y)   | overscroll-behavior: {x} {y};                               |
| sugar().overscroll({x, y}) | overscroll-behavior-x: {x};<br/>overscroll-behavior-y: {y}; |
| sugar().overscroll(value)  | overscroll-behavior-x: {value};                             |
| sugar().overscroll(value)  | overscroll-behavior-y: {value};                             |

## Usage

```tsx live
function Example() {
  const box = () =>
    sugar()
      .bg('linear-gradient(#ffbaba, #b1b7ff)')
      .px(5)
      .h(150)
      .overflowY('auto')
  const text = `The value of Pi is 3.1415926535897932384626433832795029. The value of e is 2.7182818284590452353602874713526625. Michaelmas term lately over, and the Lord Chancellor sitting in Lincolns Inn Hall. Implacable November weather. As much mud in the streets as if the waters had but newly retired from the face of the earth.`

  return (
    <>
      <div css={box().overscrollY('auto')}>{text}</div>
      <div css={sugar().h(20)} />
      <div css={box().overscroll('contain ')}>{text}</div>
      <div css={sugar().h(20)} />
      <div css={box().overscroll({ y: 'contain ' })}>{text}</div>
    </>
  )
}
```
